USAR API REAL (OPCIONAL)
========================

Se quiser usar a API Football real:

1. Abra arquivo .env
2. Mude: DEMO_MODE=true para DEMO_MODE=false
3. Salve o arquivo
4. Reinicie o servidor
5. Sistema tentará usar API real com delay de 2 segundos entre requisições

IMPORTANTE: Com API real, o sistema:
- Fará delay de 2 segundos entre requisições
- Levará mais tempo para completar (9 ligas = ~20 segundos)
- Pode receber erro 429 se a API estiver sobrecarregada
- Usará suas 100 requisições/dia do plano gratuito

PROBLEMAS COMUNS
================

Problema: "Nenhum jogo encontrado"
Solução: Verifique se DEMO_MODE=true no arquivo .env

Problema: "Erro 429"
Solução: Isso significa que a API está bloqueando
         Use MODO DEMO (DEMO_MODE=true)

Problema: "Servidor não inicia"
Solução: Verifique se MySQL está rodando
         Abra XAMPP e clique "Start" em MySQL

Problema: "Página em branco"
Solução: Aguarde 30-60 segundos
         Verifique console do navegador (F12)
