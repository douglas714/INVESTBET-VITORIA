PROBLEMAS COMUNS
================

Problema 1: Erro 429 (Too Many Requests)
- Significa que você atingiu o limite de 100/dia
- Solução: Aguarde até meia-noite UTC
- Ou: Faça upgrade para plano pago

Problema 2: Erro 404 (Not Found)
- Significa que a requisição está mal formatada
- Solução: Verifique se a API key está correta
- Ou: Reinicie o servidor

Problema 3: Erro 401 (Unauthorized)
- Significa que a API key é inválida
- Solução: Gere uma nova chave em dashboard
- Ou: Copie a chave correta

Problema 4: Sem dados retornados
- Significa que não há jogos hoje
- Solução: Tente novamente amanhã
- Ou: Use MODO DEMO para testes
