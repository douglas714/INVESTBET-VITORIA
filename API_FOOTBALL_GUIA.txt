 GUIA DE INTEGRACAO - API FOOTBALL

O QUE E API FOOTBALL?
=====================

API Football (api-football-v3.p.rapidapi.com) é uma das melhores
APIs de futebol disponíveis. Oferece:

✓ Dados de 600+ ligas de futebol
✓ Odds em tempo real
✓ Estatísticas detalhadas de times
✓ Histórico de confrontos (H2H)
✓ Standings e classificações
✓ Informações de jogadores
✓ Sem limite de requisições (plano pago)
✓ Muito mais confiável que The Odds API


POR QUE USAR API FOOTBALL?
==========================

1. Melhor Cobertura
   - The Odds API: 17 ligas
   - API Football: 600+ ligas

2. Mais Dados
   - Estatísticas completas de times
   - Histórico de confrontos
   - Informações de jogadores
   - Odds de múltiplos bookmakers

3. Melhor Confiabilidade
   - Menos erros
   - Melhor uptime
   - Suporte técnico

4. Sem Limite de Requisições
   - The Odds API: 500 requisições/mês (grátis)
   - API Football: Ilimitado (plano pago)

FUNCIONALIDADES IMPLEMENTADAS
==============================

1. Fetch Today Fixtures
   - Busca todos os jogos de hoje
   - Retorna odds em tempo real
   - Filtra apenas jogos não iniciados

2. Fetch League Fixtures
   - Busca jogos de uma liga específica
   - Próximos 7 dias
   - Com odds

3. Fetch Team Stats
   - Estatísticas de um time
   - Vitórias, empates, derrotas
   - Gols pró e contra
   - Taxa de vitória

4. Fetch Head to Head
   - Histórico de confrontos entre 2 times
   - Últimos 10 jogos
   - Resultados e gols

5. Fetch Standings
   - Classificação de uma liga
   - Posição de cada time
   - Pontos, vitórias, etc

6. Fetch Fixture Odds
   - Odds detalhadas de um jogo
   - Múltiplos bookmakers
   - Diferentes tipos de apostas

CRITERIOS DE ANALISE
====================

O robô analisa cada jogo usando 3 critérios:

1. Home Odd < 2.0
   - Time da casa é favorito
   - Odd baixa indica confiança

2. Away Odd > 3.0
   - Time visitante tem odd alta
   - Indica maior dificuldade

3. Odds Order Check
   - Home < Draw < Away
   - Ordem correta das odds

Score de Aprovação:
- Critério 1: 30 pontos
- Critério 2: 30 pontos
- Critério 3: 20 pontos
- Bonus (stats): 20 pontos
- Total: 100 pontos

Aprovado: Score >= 60

TROUBLESHOOTING
===============

P: Erro "API_FOOTBALL_KEY is not configured"?
R: Adicione API_FOOTBALL_KEY=sua_chave ao arquivo .env

P: Erro "401 Unauthorized"?
R: Verifique se a chave está correta

P: Erro "429 Too Many Requests"?
R: Você atingiu o limite de requisições da sua conta
   Upgrade para plano pago em https://dashboard.api-football.com/

P: Nenhum jogo aparece?
R: Pode não haver jogos hoje
   Tente amanhã ou use MODO_DEMO=true

P: Odds aparecem como null?
R: Nem todos os jogos têm odds disponíveis
   Sistema filtra automaticamente
