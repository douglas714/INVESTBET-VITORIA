🚀 INSTALAÇÃO RÁPIDA (3 PASSOS)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

PASSO 1: Instalar Node.js
──────────────────────────
1. Acesse: https://nodejs.org/
2. Baixe a versão LTS (recomendado)
3. Execute o instalador
4. Aceite todas as opções padrão
5. Reinicie o computador


PASSO 2: Configurar Banco de Dados (XAMPP)
────────────────────────────────────────────
1. Instale o XAMPP: https://www.apachefriends.org/
2. Abra o XAMPP Control Panel
3. Clique em "Start" ao lado de "MySQL"
4. Aguarde o status ficar verde (rodando)
5. Abra o navegador e acesse: http://localhost/phpmyadmin
6. Crie um novo banco de dados chamado: sports_betting
   - Clique em "New"
   - Digite o nome: sports_betting
   - Clique em "Create"


PASSO 3: Executar Instalador
──────────────────────────────
1. Extraia o arquivo ZIP em uma pasta (ex: C:\Projetos\)
2. Abra a pasta extraída
3. Dê DUPLO CLIQUE no arquivo: install.bat
4. Aguarde a instalação (pode levar 2-5 minutos)
5. O navegador abrirá automaticamente em http://localhost:3000
6. Aguarde 15-30 segundos para o servidor estar pronto

🎮 COMO USAR O ROBÔ
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Acesse http://localhost:3000 no navegador
2. Clique no botão "INICIAR ANÁLISE"
3. Aguarde o sistema buscar e analisar os jogos
4. Veja os resultados com status visual:
   🟡 Aguardando - Jogo ainda não começou
   🟢 GREEN - Time da casa venceu
   🔴 RED - Empate ou derrota
5. O sistema atualiza automaticamente a cada 5 minutos


🔧 CRITÉRIOS DE ANÁLISE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

O robô seleciona jogos que atendem 3 critérios obrigatórios:

1. ⭐ TIME DA CASA FAVORITO
   - O time da casa deve ter a menor odd do mercado
   - Exemplo: Casa 1.80 < Empate 3.50 < Visitante 4.20

2. 📊 ODDS EM ORDEM CRESCENTE
   - Casa < Empate < Visitante
   - Garante que o time da casa é realmente o favorito

3. 🏆 CONFRONTO DIRETO
   - Time da casa venceu TODOS os últimos jogos
   - Vitória deve ser por 2+ gols de diferença
   - Garante histórico positivo


🎯 SELEÇÃO DOS MELHORES JOGOS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

O sistema seleciona os 6 melhores jogos baseado em:

1. Menor odd do time da casa (favorito mais forte)
2. Maior diferença entre odd da casa e visitante (maior confiança)

Exemplo:
  🥇 Jogo A: Casa 1.50 (Diferença: 2.70)
  🥈 Jogo B: Casa 1.65 (Diferença: 2.50)
  🥉 Jogo C: Casa 1.80 (Diferença: 2.40)

🔧 SOLUÇÃO DE PROBLEMAS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

ERRO: "Node.js não encontrado"
➜ Solução: Instale o Node.js 18+ e reinicie o computador

ERRO: "Dependências não encontradas"
➜ Solução: Execute install.bat primeiro

ERRO: "Cannot connect to database"
➜ Solução:
  1. Verifique se o MySQL está rodando (XAMPP Control Panel)
  2. Confirme que o banco "sports_betting" existe
  3. Edite o arquivo .env e corrija DATABASE_URL se necessário

ERRO: "Porta 3000 em uso"
➜ Solução: Feche outros programas usando a porta 3000

ERRO: "API Key inválida"
➜ Solução:
  1. Verifique se o arquivo .env existe
  2. Confirme que tem a linha: THE_ODDS_API_KEY=3a50d923ce7f50b649c36ced65029efd
  3. Salve o arquivo e reinicie (feche e execute start.bat novamente)

ERRO: "Nenhum jogo encontrado"
➜ Solução:
  1. Verifique se há jogos de futebol hoje
  2. Aguarde alguns minutos e tente novamente
  3. Verifique a conexão com a internet

4. Salve o arquivo
5. Execute start.bat novamente
